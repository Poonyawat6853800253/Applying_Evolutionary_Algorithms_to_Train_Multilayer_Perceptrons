# Applying_Evolutionary_Algorithms_to_Train_Multilayer_Perceptrons
This notebook was developed to demonstrate an alternative way of training Multilayer Perceptrons (MLPs) using evolutionary algorithms instead of standard backpropagation. The main goal is to compare how different population-based optimization methods can search for suitable neural network weights and biases.

The notebook implements and evaluates four algorithms: Binary Genetic Algorithm (BGA), Real-Coded Genetic Algorithm (RGA), Basic Differential Evolution (DE), and Adaptive Differential Evolution (ADE). These methods are tested on both classification and regression problems, with results compared using MSE loss, accuracy, and execution time across multiple runs.

This project was made for learning, experimentation, and future extension in evolutionary neural networks. It can be used as a reference for students studying optimization, machine learning, and metaheuristic approaches, and it can also serve as a starting point for improving model design, parameter tuning, and benchmarking on more complex datasets.
